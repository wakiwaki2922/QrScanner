# QR Scanner

Quét mã QR từ màn hình máy tính.

## Tải về

Download file `.exe` từ [Releases](../../releases)

## Tính năng

- Quét QR code từ màn hình
- Tự động copy vào clipboard
- Giao diện đơn giản

## Build

```bash
build.bat
```

## Chạy từ source

```bash
pip install -r requirements.txt
python qr_scanner_app.py
```
